# Core Java Labs

A comprehensive collection of Java programming exercises covering fundamental to advanced concepts, along with MongoDB query practice labs.

## Project Structure

```
aakash-joshi-core-java-labs/
├── Java Labs/          # Core Java programming exercises (Lab 1-9)
└── MongoDB Labs/       # MongoDB query practice notebooks
```

## Java Labs Overview

The Java Labs section contains 9 labs covering:

- **Lab 1**: Basic Java operations (digit manipulation, calculations)
- **Lab 2**: Array operations and algorithms
- **Lab 3**: String manipulation with StringTokenizer
- **Lab 4**: Object-oriented programming and inheritance
- **Lab 5**: Exception handling and custom exceptions
- **Lab 6**: Collections framework (HashMap, ArrayList)
- **Lab 8**: Multithreading and file I/O
- **Lab 9**: Executor framework and concurrent programming

### Additional Projects
- **Employee Insurance System** (`com.cg.eis`): A layered application demonstrating service-oriented architecture with employee management and insurance scheme assignment.

## MongoDB Labs

See the [MongoDB Labs README](./MongoDB%20Labs/README.md) for detailed information about MongoDB query exercises.

## Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Eclipse IDE or any Java IDE
- MongoDB (for MongoDB Labs)

### Running Java Labs
1. Import the project into your IDE
2. Navigate to the specific lab exercise
3. Run the main class for each exercise
